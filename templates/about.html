About SkyCast

SkyCast is a lightweight, server-rendered weather dashboard built entirely with Flask, Jinja2, and hand-written CSS — no JavaScript frameworks required. Live conditions and forecasts are sourced from the free Open-Meteo API.

Every page you see — Dashboard, Forecast, Settings, and this About page — is a genuine Flask route rendering its own template, not a single page with scroll anchors. That means each section works independently, even if a weather lookup happens to be unavailable.
